POST /token
Generate an authentication token required for API access.

Response

{ "token": "eyJhbGciOiJIUzI1..." }
POST /search
Search for tracks, albums, playlists, or artists.

Parameters

query (string): Search query
search_type (string): One of 'track', 'album', 'artist', 'playlist'
limit (integer, optional): Maximum number of results (default: 10)
POST /top-tracks
Get an artist's top tracks.

Parameters

artist_name (string): Name of the artist
limit (integer, optional): Maximum number of tracks (default: 10)

Rate Limits

The API is limited to 100 requests per token per minute. After exceeding this limit, requests will return a 429 status code.
